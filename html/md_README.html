Welcome to ChessMind

When it was recommended that I partake in the Purdue University Post Graduate program in Data Science, (with it's focus on Machine Learning and AI), a curiosity developed concerning the ability of a mere LinearRegression model and the ability to predict Gary Kasparov's next move. This curiosity grew into this project. >GIVEN that advanced AI software platforms have already been developed for Chess games >WHEN experimentation on Machine Learning and AI with respect to Chess logic gets the better of me

>THEN we add popular methods explored in the field of Machine Learning and AI with this C++/Python open source package.
